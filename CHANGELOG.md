# Changelog

## 1.4.0 - 2026-04-07

- 記録の詳細モーダルを追加（タイムライン・ログカードから直接開けるように）
- コンパクトタイムラインのログチップから詳細・編集モーダルを開けるように
- キーボードショートカット: Nキーで記録作成モーダルを開く、Ctrl+Enterで保存
- FAB ホバー時に N キーのショートカットヒントをツールチップで表示
- 活動ページで追加・編集・アーカイブ後に一覧を即時反映

## 1.3.0 - 2026-04-02

- **type**: MINOR
- **title**: ライトモード・スター記録・統計ページ刷新

### 変更内容

#### ライトモード対応

- ライト / ダークモードの切り替えに対応
- システム設定への自動追従に加え、アプリ内からも手動で切り替え可能

#### 余韻 → スター記録に刷新

従来の「余韻（Reflection）」機能を、**スター数による評価**システムへ刷新。

- 記録ごとに 1〜5 のスターで強度・達成感を表現できるように変更
- 記録の作成・編集を 1 つのモーダルに統合し、操作を簡略化

#### ナビゲーションと統計の再構成

| Before | After |
|--------|-------|
| ホーム / 月次ページ（別画面） | ホームにタブを追加（月次ページ廃止） |
| 統計なし（Activity 詳細内） | 統計を独立ページとして新設 |

- ナビゲーションの名称を整理し、導線をシンプルに
- 統計ページのカテゴリカードから Activity 詳細へ直接遷移可能に
- Activity 一覧のコンパクトビューを時刻昇順に変更

#### 操作性の改善

- 記録の日時に**未来の日時も設定可能**に（制限を解除）
- タイムラインのレイアウトをよりコンパクトに調整（スター表示・ノート行を分離）
- `formatCompactTime` を拡張し、今日 / 昨日 / それ以前を適切に表示

#### その他

- 設定ページのスペーシングを調整
- ページヘッダーのサイズ・余白を統一

# Changelog

## 1.4.0 - 2026-04-07

### 追加

- 記録の詳細モーダルを追加し、タイムラインやログカードから直接開けるようになりました
- N キーで記録作成モーダルを開く、Ctrl+Enter で保存するキーボードショートカットを追加しました
- FAB ホバー時に N キーのショートカットヒントをツールチップで表示するようになりました

### 改善

- コンパクトタイムラインのログチップから詳細・編集モーダルを直接開けるようになりました
- 活動ページで追加・編集・アーカイブ後に一覧が即時反映されるようになりました

## 1.3.0 - 2026-04-02

- **type**: MINOR
- **title**: ライトモード・スター記録・統計ページ刷新

### 追加

- ライト / ダークモードの切り替えに対応しました。システム設定への自動追従に加え、アプリ内から手動でも切り替えられます
- 記録ごとに 1〜5 のスターで強度・達成感を表現できるスター評価に刷新しました（従来の「余韻」機能を置き換え）
- 統計を独立ページとして新設し、カテゴリカードから Activity 詳細へ直接遷移できるようになりました

### 改善

- 記録の作成・編集を 1 つのモーダルに統合し、操作を簡略化しました
- ホームにタブを追加して月次ページを統合しました（月次ページは廃止）
- ナビゲーションの名称を整理し、導線をシンプルにしました
- Activity 一覧のコンパクトビューを時刻昇順に変更しました
- 記録の日時に未来の日時も設定できるようになりました
- タイムラインのレイアウトをよりコンパクトに調整しました

## 1.2.0 - 2026-03-24

### 追加

- 月次集計ページをナビゲーションに独立して追加しました
- Activity に説明文フィールドを追加しました
- Home にアクティビティ・メモの検索フィルターを追加しました

### 改善

- Home をタイムライン形式にリニューアルし、FAB から直接記録できるようになりました（記録ページを廃止）
- Activity 一覧をドラッグ＆ドロップで並び替えられるようになりました
- Activity 一覧のメニューを廃止し、統計・編集・アーカイブボタンを独立配置しました
- Activity 詳細の最近の記録をカード形式に変更しました
- 記録の日時を Twitter 風の相対時間で表示するようになりました
- タイムライン・カードのデザインと余白を調整しました

### 修正

- 記録の作成・削除・日時編集後にホームのデータが即時反映されるよう修正しました

## 1.1.0 - 2026-03-22

### 追加

- History にタイムライン表示モードを追加しました
- ログ作成時にメモを入力できるようになりました
- Home「最近のピーク」にページネーションを追加しました
- 新規ユーザーにデフォルト Activity を自動作成するようになりました
- 各ページに説明文を追加しました

### 改善

- Activity 詳細の最近の記録を日付グループ表示に変更し、30件まで表示されるようになりました
- ログカード・ActivityItem をコンパクトなデザインに整理しました
- 日付選択 UI を作成・編集で統一しました
- セッション有効期限を 90 日に延長し、ログインページにオートフィル対応を追加しました
- 月次統計への導線を改善しました

### 修正

- ログカードのメモ改行が正しく表示されるようになりました
